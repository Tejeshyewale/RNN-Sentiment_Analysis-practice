# Sentiment Analysis using SimpleRNN

## 📌 Project Overview

This project implements a **Sentiment Analysis model** using a **Simple Recurrent Neural Network (SimpleRNN)**. The model is trained to classify text data (such as reviews or sentences) into different sentiment categories like **positive** or **negative**.

This project demonstrates the application of **Natural Language Processing (NLP)** and **Deep Learning** techniques using neural networks.

---

## 🚀 Features

* Text preprocessing and cleaning
* Integer encoding of text data
* Sequence padding for uniform input size
* SimpleRNN model implementation
* Model training and evaluation
* Sentiment prediction

---

* TensorFlow / Keras

1. Load dataset
2. Clean and preprocess text
3. Tokenize and perform integer encoding

5. Build SimpleRNN model
6. Train the model
7. Evaluate performance
